音频解码和播放流程:
输入 Opus 编码数据
  ↓
Opus 解码
  → 使用 opus 解码器
  → 解码为 16 位 PCM 数据
  → 处理解码失败情况(生成静音帧)
  ↓
音频缓冲管理
  → 将解码后的 PCM 数据放入缓冲队列
  → 监控缓冲区大小(默认上限100帧)
  → 缓冲区满时丢弃旧数据
  ↓
音频播放控制
  → 等待缓冲累积(默认10帧)
  → 获取系统默认音频输出设备
  → 适配设备支持的配置(采样率/声道)
  ↓
创建音频输出流
  → 设置回调处理音频数据
  → 从缓冲区读取 PCM 数据
  → 转换为设备支持的格式
  → 填充静音(缓冲区空时)
  ↓
播放监控
  → 检测数据流状态
  → 自动停止(无数据超过1秒)
  → 清理资源并回调
